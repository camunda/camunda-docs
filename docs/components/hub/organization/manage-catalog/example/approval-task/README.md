## Approval Task

A standardized approval user task with built-in escalation and delegation support.

### Capabilities

- Single-approver and multi-approver workflows
- Automatic escalation after a configurable timeout
- Delegation to a substitute approver

### Usage

Apply this template to a **User Task** in your BPMN diagram. Configure the approver group and escalation timeout in the properties panel.
