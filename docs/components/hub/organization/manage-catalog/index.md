## About the catalog

In Camunda Hub, the catalog is a collection of reusable automation assets, such as element templates. Catalog assets live in your own Git repository and are uploaded to Camunda Hub, where:

- Center of excellence teams assess and approve them for use within the organization.
- Delivery teams discover and use the assets when they model business processes.
