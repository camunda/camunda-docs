Desktop Modeler can directly deploy diagrams and start process instances in Camunda 8. Follow the steps below to deploy a diagram to Camunda 8 Orchestration Clusters.

1. Ensure you have already setup and selected a connection by following the [Connect to Camunda 8 guide](./connect-to-camunda-8.md).

   ![dev cluster connected](./img/connection-selector-dev-connection.png)

2. Click the rocket-shaped deployment icon:

   ![deploy icon](./img/deploy-icon.png)

3. Click **Deploy** to perform the actual deployment.

   ![deploy diagram](./img/deploy-diagram.png)

4. If the deployment is successful, you will see a confirmation message:

   ![deployment successful](./img/deploy-diagram-success.png)

:::note
As a next step, [start a new process instance](./start-instance.md).
:::
