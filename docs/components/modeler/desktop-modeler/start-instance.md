---
id: start-instance
title: Start a new process instance
description: "After you have deployed your process to Camunda 8, you can start a new instance of this process via the play icon."
---

1. Ensure you have already setup and selected a connection by following the [Connect to Camunda 8 guide](./connect-to-camunda-8.md):

   ![dev cluster connected](img/connection-selector-dev-connection.png)

2. You can start a new instance of this process via the play icon:

   ![start instance icon](./img/start-instance-icon.png)

3. Start the process instance with optional variables.

   ![start instance successful](./img/start-instance.png)
   After the instance was started successfully, a corresponding message is displayed.

   ![start instance successful](./img/start-instance-successful.png)
