This reference architecture describes how to deploy the [Camunda 8 Orchestration Cluster](./reference-architecture.md#orchestration-cluster-vs-web-modeler-and-console) to Amazon ECS with Fargate and Aurora PostgreSQL. This deployment method is ideal for users who want a portable, consistent runtime and the benefits of containerization without managing Kubernetes.
