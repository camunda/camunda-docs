## Installation and configuration

Camunda Hub is a single platform that empowers you to operate efficiently, scale safely, and build with confidence.

For teams responsible for establishing standards, infrastructure, and guardrails for automation at scale, it provides tools to:

- Manage organizational, infrastructural, and collaborative resources
- Govern the use of reusable assets
- Monitor operations and assess business value

For teams responsible for building and delivering processes, it provides tools to:

- Discover and use approved reusable assets
- Model and deploy business processes

In this section, you will learn how to install, and configure Camunda Hub in your Self-Managed environment.
