:::note Helm CLI v4 required

Camunda 8.10 and later (chart 15.x and later) require the Helm CLI v4. Helm v3 is not supported. If you are switching from the Helm v3 CLI, no release-state migration is required. See [Move from the Helm v3 CLI to v4](/self-managed/deployment/helm/operational-tasks/moving-helm-v3-to-v4.md).

:::
