Search for activatable activities within ad-hoc sub-processes based on given criteria.

Note that this API currently requires filters for both process definition key and ad-hoc
sub-process ID and does not support paging or sorting.

:::note
This endpoint is an [alpha feature](/components/early-access/alpha/alpha-features.md) and may be subject to change
in future releases.
:::
