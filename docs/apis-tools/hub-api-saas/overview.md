## Authentication

See the [authentication guide](/apis-tools/hub-api-saas/authentication.md) for setup instructions.

## Migrating from Web Modeler API v1

Web Modeler API v1 is deprecated in Camunda 8.10 and will be removed in 8.12. [Migrate](/apis-tools/hub-api-saas/overview.md) to Camunda Hub REST API v2 before upgrading to 8.12.
