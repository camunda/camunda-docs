Use the official TypeScript SDK to build Camunda 8 applications in TypeScript.

## Get started

Choose one of the following options to get started with Camunda 8 TypeScript development:

| SDK/package                                                                                                                                                                 | Description                                                                                                    |
| --------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------- |
| [TypeScript SDK](camunda8-sdk.md)<br/>[`@camunda8/sdk`](https://www.npmjs.com/package/@camunda8/sdk)                                                                        | Provides clients for all Camunda 8 APIs. Use it in Node.js environments.                                       |
| [Orchestration Cluster API TypeScript client](oca-client.md)<br/>[`@camunda8/orchestration-cluster-api`](https://www.npmjs.com/package/@camunda8/orchestration-cluster-api) | A lightweight client for the Camunda 8.8+ Orchestration Cluster REST API. Use it in Node.js or in the browser. |
