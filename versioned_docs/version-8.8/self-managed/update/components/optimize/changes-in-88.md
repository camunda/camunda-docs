This page summarizes notable changes introduced in Optimize 8.8 that may affect your upgrade.

If you are upgrading from Optimize 8.7, follow the steps in [Upgrade Optimize from 8.7 to 8.8](./870-to-880.md).

The upgrade to 8.8 is supported from any 8.7 release. If you are upgrading from an earlier version, see [Upgrading from an earlier Optimize version](./index.md#upgrading-from-an-earlier-optimize-version).

## Changes in supported environments

### Database requirements

Optimize 8.8 supports the following database versions:

- Elasticsearch 8.16 or later
- Amazon OpenSearch 2.17 or later

Before upgrading Optimize, make sure your database cluster is running a supported version.

For the full list of supported versions and components, see [supported environments](/reference/supported-environments.md#component-requirements).
