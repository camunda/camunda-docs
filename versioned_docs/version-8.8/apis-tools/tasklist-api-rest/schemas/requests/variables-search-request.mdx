### Fields

#### [<code style={{ fontWeight: 'normal' }}>VariablesSearchRequest.<b>variableNames</b></code>](#)<Bullet />`string` <Badge class="secondary" text="list"/>

> Name of variables to find.

#### [<code style={{ fontWeight: 'normal' }}>VariablesSearchRequest.<b>includeVariables</b></code>](#)<Bullet />[`IncludeVariable`](../models/include-variable.mdx) <Badge class="secondary" text="list"/>

> An array used to specify a list of the variables that should be included in the response.

### Notes

> `VariablesSearchRequest.variableNames` and `VariablesSearchRequest.includeVariables` cannot be used together in the same API request.

> It is recommended to use `VariablesSearchRequest.includeVariables`, as `VariablesSearchRequest.variableNames` will be deprecated in a future release.
