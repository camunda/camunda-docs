Variable used in task.

```graphql
type Variable {
  id: ID!
  name: String!
  value: String!
  previewValue: String!
  isValueTruncated: Boolean!
}
```

### Fields

#### `id` ([`ID`](/docs/apis-tools/tasklist-api/scalars/id))

#### `name` ([`String`](/docs/apis-tools/tasklist-api/scalars/string))

#### `value` ([`String`](/docs/apis-tools/tasklist-api/scalars/string))

full variable value

#### `previewValue` ([`String`](/docs/apis-tools/tasklist-api/scalars/string))

value preview (limited in size)

#### `isValueTruncated` ([`Boolean`](/docs/apis-tools/tasklist-api/scalars/boolean))

shows, whether previewValue contains truncated value or full value
