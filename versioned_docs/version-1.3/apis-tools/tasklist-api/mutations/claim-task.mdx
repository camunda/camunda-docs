Claim a task with taskId to currently logged in user. Returns the task.

```graphql
claimTask(
  taskId: String!
  assignee: String
): Task!

```

### Arguments

#### `taskId` ([`String`](../scalars/string.mdx))

#### `assignee` ([`String`](../scalars/string.mdx))

### Type

#### [`Task`](../objects/task.mdx)

Describes the User task.
