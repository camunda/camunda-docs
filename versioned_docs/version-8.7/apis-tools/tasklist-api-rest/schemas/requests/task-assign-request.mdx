### Fields

#### [<code style={{ fontWeight: 'normal' }}>TaskAssignRequest.<b>assignee</b></code>](#)<Bullet />`string`

> When using a `JWT` token, the assignee parameter is NOT optional when called directly from the API. The system will not be able to detect the assignee from the JWT token, therefore the assignee parameter needs to be explicitly passed in this instance.

#### [<code style={{ fontWeight: 'normal' }}>TaskAssignRequest.<b>allowOverrideAssignment</b></code>](#)<Bullet />`boolean`

> When `true` the task that is already assigned may be reassigned again. Otherwise, the task must be first unassigned and only then assigned again. (Default: `true`).
