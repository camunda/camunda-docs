---
id: start-instance
title: Start a new process instance
description: "After you have deployed your process to Camunda 8, you can start a new instance of this process via the play icon."
---

1. Ensure you have already set up and selected a connection by following the [Connect to Camunda 8](./connect-to-camunda-8.md) guide.

   ![dev cluster connected](img/connection-selector-dev-connection.png)

2. Click the play icon to start a new process instance.

   ![start instance icon](./img/start-instance-icon.png)

3. Enter optional variables for starting the process instance. Click **Start BPMN process instance** to deploy and start the instance.

   ![start instance overlay](./img/start-instance.png)

   A confirmation message confirms that the instance started successfully.

   ![start instance successful](./img/start-instance-successful.png)
