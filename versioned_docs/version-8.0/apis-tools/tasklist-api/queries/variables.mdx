```graphql
  taskId: String!

): [Variable!]!

```

### Arguments

#### `taskId` ([`String`](../scalars/string.mdx))

#### `variableNames` ([`String`](../scalars/string.mdx))

### Type

#### [`Variable`](../objects/variable.mdx)
