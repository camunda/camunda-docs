# Agent instructions for camunda-docs

This file provides instructions for AI coding agents (Claude Code, OpenAI Codex, and similar tools) working autonomously in this repository.

## Style and content guidelines

Follow these instructions and associated instruction files. They apply to all AI agents:

- **Start here:** `.github/copilot-instructions.md`.
- **Content rules (Markdown files):** `.github/instructions/content.instructions.md` - language, grammar, formatting, links, and product terminology.
- **Repo rules (all files):** `.github/instructions/repo.instructions.md` — file naming, repo structure, PR workflow, and commit conventions.
- **Full style guide:** `howtos/technical-writing-styleguide.md`.

Read these before making any changes. They are the source of truth for contribution standards.
